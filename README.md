# 📊 Dashboard OSN-K Dikdas

Interactive web dashboard untuk menampilkan dan menganalisis data peserta **OSN-K (Olimpiade Sains Nasional Tingkat Kabupaten/Kota)** jenjang **SD & SMP**.

---

## ✨ Fitur Utama

- 🔍 Pencarian data peserta (nama & sekolah)
- 🎯 Filter berdasarkan jenjang (SD / SMP)
- 🏆 Highlight Top 3 peserta
- 📊 Visualisasi data sekolah (Chart.js)
- 📄 Tabel data dengan pagination
- 🌙 Dark Mode (Mode Gelap)
- 📥 Export data ke Excel

---

## 🛠️ Teknologi

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Chart.js
- SheetJS (xlsx)

---

## 🚀 Cara Menjalankan

⚠️ **Penting:** Jangan buka langsung file HTML (file://), karena data JSON tidak akan terbaca.

### ✅ Gunakan Live Server

1. Install extension **Live Server** di VS Code
2. Klik kanan `index.html`
3. Pilih **Open with Live Server**

---

## 📌 Catatan

- Pastikan nama file sesuai:
  - `script.js`
  - `style.css`
  - `data.json`
- Pastikan tidak ada error di Console (F12)
- Gunakan browser modern (Chrome, Edge, Firefox)
